# 🎉 EVENZA – Smart College Event Management System

EVENZA is a modern web-based platform designed to simplify the process of managing and participating in college events.  
It allows users to explore events, register seamlessly, and helps organizers manage event data efficiently.

---
## 📌 Features

✨ Dynamic Event Listings  
- Events displayed as interactive cards  
- Real-time updates using Firebase  

📝 Event Registration  
- Simple and user-friendly registration form  
- Data stored in real-time database  

🤖 Chatbot Support  
- Basic chatbot for user guidance  
- Enhances user interaction  

🎯 Clean UI/UX  
- Responsive and visually appealing design  
- Smooth navigation experience  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Firebase (Firestore / Realtime DB)  
- **Hosting:** GitHub Pages  

---

## 🔐 Security Implementation

- Public read access for events  
- Restricted write access to prevent unauthorized modifications  
- Controlled registration submissions  
- Firebase security rules applied for safe data handling
